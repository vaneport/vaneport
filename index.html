Educación

Grado Superior en Desarrollo de Aplicaciones Multiplataforma

Como estudiante de desarrollo de aplicaciones multiplataforma, he adquirido una base y práctica en una variedad de tecnologías y lenguajes de programación. Mis estudios se han centrado en:

  • Desarrollo front-end utilizando HTML y CSS para crear interfaces de usuario limpias y responsivas.
  • Programación en Java y manejo de bases de datos SQL y no SQL
  • Implementación de aplicaciones web seguras con Spring Boot y Spring Security.
  • Creación de aplicaciones móviles usando Jetpack Compose y Kotlin, enfocándome en una experiencia de usuario intuitiva y moderna.
  • Aplicación de patrones de diseño y persistencia de datos con Hibernate.
  • Integración de servicios en la nube como Dropbox y Firebase para mejorar la funcionalidad y el almacenamiento de las aplicaciones.
